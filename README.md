# AI Usage Monitor (Chrome 拡張機能)

Claude Pro や Gemini Advanced、Google AI Studio の送信数制限をスマートに監視・記録し、ビジュアル表示する Manifest V3 準拠の Chrome 拡張機能です。

個人データやパスワード、セッショントークンなどは一切不要・保存しないため、**完全にローカルで動作する設計**になっています。

---

## 🌟 主な機能

1. **送信アクションの自動検知**:
   * Claude.ai、Gemini (gemini.google.com)、Google AI Studio でプロンプトを送信すると、送信ボタンクリックまたはEnterキー押下（日本語IMEの確定Enterは無視する安全設計）を自動で検知しカウントします。
2. **ローリング・ウィンドウ（時間枠）方式の自動減衰**:
   * メッセージ送信のタイムスタンプを保存し、設定した時間（Claudeなら5時間、Geminiなら3時間）が経過すると古い履歴から自動的に消去され、残り回数がリアルタイムで回復します。
3. **すりガラス風（グラスモーフィズム）の floating バー**:
   * AIのWebページ上にドラッグ可能な美しいステータスバーを自動インジェクションします。
   * バーには現在の利用状況が円形インジケーターの色（安全＝緑、警戒＝黄、上限間近＝赤）で表示され、マウスホバーで「+1 / -1」のクイック微調整や「次の枠解放までのカウントダウン」を表示します。
4. **一括管理ダッシュボード（ポップアップ）**:
   * ブラウザの拡張機能アイコンをクリックすると、3つのサービスすべての状態が一目でわかる、美しいダッシュボードが開きます。
5. **詳細な送信履歴の確認と編集**:
   * ダッシュボード下部を展開すると、本日の送信タイムスタンプ（例: `15:32:01（12分前）`）が新しい順にリスト表示されます。誤検知された場合は、ゴミ箱ボタンから特定の送信履歴を個別に削除することができます。
6. **完全カスタマイズ設定**:
   * 各AIサービスの上限数（回）や時間枠（時間）を簡単に編集可能（デフォルトに戻す機能付き）。

```
│   └── popup.js                   # 進捗アニメーション、手動調整、設定管理ロジック

│   ├── content.js                 # 送信検知 & ドラッグ可能な画面ウィジェットの制御
│   └── widget.css                 # 画面内ウィジェットのプレミアムCSS
└── README.md                      # このドキュメント
```

## 🚀 インストール方法

1. **Google Chrome** を開き、アドレスバーに `chrome://extensions` と入力してエンターキーを押します（拡張機能一覧画面が開きます）。
2. 画面右上にある **「デベロッパー モード」** のトグルスイッチを **オン** にします。
3. 画面左上に出現する **「パッケージ化されていない拡張機能を読み込む」** ボタンをクリックします。
4. ファイル選択ダイアログが表示されるので、ダウンロードした（あるいはクローンした）`ai-usage-monitor` フォルダを選択して開きます。

これでインストールは完了です！ブラウザのツールバー（パズルのピースアイコン）から「AI Usage Monitor」のピン留めを行ってください。

## 💡 使い方とTIPS

* **ウィジェットの移動**: AIのWebサイトの右下に出現するバーの上部（ドットが並んでいる箇所）をドラッグすることで、画面上の好きな位置へ自由に移動できます。位置は自動保存され、ページをリロードしても維持されます。
* **手動での校正**: AI側の送信規制が早まったり、自動カウントがズレた場合は、ウィジェットの「-1」や「+1」ボタン、またはダッシュボードから手動で自由に修正できます。
* **日本語入力時の配慮**: `content.js` は日本語変換時の「確定Enter」を自動判別し、変換確定のためのEnterではカウントされない設計になっているため安心です。
